## Install

https://docs.n8n.io/integrations/community-nodes/installation/

1. Go to Settings (Cogwheel)
2. Click on "Community Nodes"
3. Enter "n8n-nodes-close" into the text box
4. Click on "I understand the risk ..."
5. Click on "Install"
